# 💳 Payment Processing System

System Design Project – Scalable Payment Infrastructure

---

# Problem Statement

Design a payment processing system that allows users to make secure payments through credit cards or digital wallets.

The system must handle transactions reliably and maintain transaction history.

---

# Functional Requirements

• User initiates payment  
• Validate payment details  
• Process payment request  
• Store transaction record  
• Return payment status  

---

# Non Functional Requirements

• High reliability  
• Secure transactions  
• Low latency processing  
• Scalable architecture  

---

# Core Concepts

• Payment authorization  
• Fraud detection  
• Transaction logging  
• Service orchestration  

---

# High Level Architecture

Client Application
 ↓
API Gateway
 ↓
Payment Service
 ↓
Fraud Detection Service
 ↓
Transaction Service
 ↓
Database
